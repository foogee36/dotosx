# dotosx

## Setup

Initialize with chezmoi:
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/foogee36/dotosx/refs/heads/main/install.sh)"
```

Restart terminal.

Install commands:
```
chezmoi cd
make setup
```
